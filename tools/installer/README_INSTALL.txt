Mini Mixer Overlay - Paket
==========================

Schnellstart:
1) Install.bat ausfuehren
2) App startet danach automatisch

Manuell starten:
- Start.bat

Deinstallieren:
- Uninstall.bat

Installationsort (Standard):
- %LOCALAPPDATA%\Programs\MiniMixerOverlay

Hinweise:
- Falls eine alte, haengende Instanz vorhanden ist, einmal Windows neu starten.
- Einstellungen bleiben unter %APPDATA%\MiniMixerOverlay\settings.json erhalten.
